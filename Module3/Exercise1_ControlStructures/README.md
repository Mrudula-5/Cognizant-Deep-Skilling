# Exercise 1: Control Structures

## Objective
Implement PL/SQL control structures for banking operations.

## Scenarios

### Scenario 1
Applied a 1% discount on loan interest rates for customers above 60 years.

### Scenario 2
Updated the IsVIP flag to TRUE for customers whose balance is greater than 10000.

### Scenario 3
Displayed reminder messages for customers whose loans are due within the next 30 days.

## Concepts Used
- PL/SQL
- BEGIN-END Block
- FOR LOOP
- UPDATE
- COMMIT
- JOIN
- DBMS_OUTPUT.PUT_LINE
- SYSDATE

## Conclusion
Successfully implemented PL/SQL control structures to automate banking operations.
